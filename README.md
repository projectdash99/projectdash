## Overview

DashChat is a state-of-the-art, premium AI chat platform designed to provide a seamless and highly responsive user experience. It serves as a unified workspace that allows users to interact with multiple AI models (Gemini, DeepSeek, Claude, GPT, and Grok) from a single beautifully crafted interface. 

With a focus on radical calm and deep work, DashChat takes inspiration from the best modern chat interfaces, bringing forward an editorial typography scale, fluid micro-interactions, and a sophisticated Apple-inspired aesthetic that prioritizes accessibility and progressive disclosure.

## Key Features

- **Multi-Model Support**: Effortlessly switch between top-tier AI models directly from the interface.
- **Premium Design System**: Uses carefully selected web fonts (Inter for UI, Fraunces for editorial headings), curated dark/light modes, and a custom ambient gradient orb for branding.
- **Responsive Architecture**: Fully optimized across all breakpoints. Features sliding sidebars on tablets, bottom sheet model selectors on mobile, and safe-area constraints for iOS.
- **Framer Motion Animations**: Fluid and natural interactions, including message slide-ins, hover lifts, and tactile button scales.
- **Seamless Authentication**: Powered by Supabase for secure and rapid user login and session management.
- **Accessible Interactions**: Complete keyboard navigation support, WCAG AA contrast compliance, and comprehensive screen-reader aria labels.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS v4 & shadcn/ui principles
- **Animations**: Framer Motion
- **Database & Auth**: Supabase
- **Icons**: Lucide React
- **AI Integration**: Vercel AI SDK
